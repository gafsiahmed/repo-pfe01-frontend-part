# Repo PFE01 — Frontend Part  
**Repo pour le code de chaque séance du Cohorte 01/25 Préparation PFE — Partie Frontend**

## Description du projet  
Ce dépôt contient tout le code front-end développé durant les séances de la formation/cohorte « Préparation PFE 01/25 ».  
Chaque séance est séparée dans un dossier (seance1, seance2, etc.), ce qui permet de suivre l’évolution pas-à-pas du projet — idéal pour vos révisions, pour revoir le code écrit en classe, ou pour partager facilement les travaux.  

L’objectif est de fournir aux étudiants une trace claire et organisée de ce que nous avons appris lors de chaque session. Ce dépôt servira aussi de référence pour vos projets futurs.
