# Repo PFE01 — Frontend Part

**Repo pour le code de chaque séance du Cohorte 01/25 — Préparation PFE (Partie Frontend)**

## 📘 Description du projet

Ce dépôt contient tout le code front-end développé durant les séances de la formation/cohorte « Préparation PFE 01/25 ».
Chaque séance est organisée dans un dossier séparé (`seance1`, `seance2`, etc.), ce qui permet de suivre l’évolution du contenu pas à pas — idéal pour réviser, revoir le code écrit en classe ou partager facilement les travaux.

L’objectif est de fournir aux étudiants une trace claire, propre et organisée de ce que nous avons appris lors de chaque session. Ce dépôt pourra également servir de référence pour vos futurs projets.

---

## 🚀 Comment utiliser / exécuter le code

### 1. Cloner le dépôt

```bash
git clone https://github.com/gafsiahmed/repo-pfe01-frontend-part.git
cd repo-pfe01-frontend-part
```

### 2. Récupérer les nouvelles séances

```bash
git pull
```
